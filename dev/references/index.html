Javis.latex — Method
latex(t::LaTeXString, font_size::Real, action::Symbol)

Add the latex string text at the current position. The current position is the top left corner of the latex path. This only works if tex2svg is installed. It can be installed via:

npm install -g mathjax-node-cli

The default font_size is 10 and the default action is :stroke.
